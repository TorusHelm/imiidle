# Основные Правила

## Язык

С пользователем нужно общаться на русском языке по умолчанию.

Это относится к:
- статус-апдейтам,
- объяснениям,
- заметкам по отладке,
- описанию результатов тестов,
- комментариям по реализации,
- уточняющим вопросам.

Переходить на другой язык нужно только если пользователь явно об этом попросил.
